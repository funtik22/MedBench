# RuMedTop3 Telegram Bot

Telegram-бот для подбора **ТОП-3 возможных диагнозов по классификации МКБ-10**
на основе текстового описания симптомов.  
Бот использует ML-модель, обученную в рамках задачи **RuMedTop3**.

---

## Описание

Бот принимает текст на русском языке с описанием симптомов пациента и возвращает:
- ТОП-3 наиболее вероятных медицинских кодов (МКБ-10)
- Расшифровку кодов в виде названий заболеваний

---

## Возможности
- Поддержка русского языка
- Предсказание ТОП-3 диагнозов
- Использование ансамбля моделей
- Расшифровка кодов МКБ-10
- Быстрый ответ

---

## Используемая модель
- TF-IDF (char + word n-grams)
- Ансамбль моделей:
  - Logistic Regression
  - LinearSVC (с калибровкой вероятностей)
  - Multinomial Naive Bayes
  - Random Forest
- Rank-based ансамблирование
- Метрики на Dev:
  - Accuracy@1: **50.00%**
  - Hit@3: **72.63%**

## Пример ответа

ТОП-3 возможных диагноза:

1. J06.9 — Острая инфекция верхних дыхательных путей неуточненная
2. J00 — Острый назофарингит (насморк)
3. J20 — Острый бронхит

Результат носит справочный характер и не является медицинским заключением.
